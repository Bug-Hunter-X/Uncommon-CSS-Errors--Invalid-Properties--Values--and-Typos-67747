# Uncommon CSS Errors
This repository demonstrates some uncommon CSS errors that can be tricky to debug, focusing on invalid properties, unsupported values, and typos.

## Bug.css
The `bug.css` file contains the CSS code with errors. These errors can manifest in different ways depending on the browser and its CSS engine.  The code might result in no visual effect, unexpected styles, or even browser errors.

## bugSolution.css
The `bugSolution.css` file shows the corrected CSS code with explanations of the fixes implemented.

Feel free to contribute by adding more examples of uncommon CSS errors and their solutions.
